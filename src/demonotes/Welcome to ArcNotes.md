# Welcome to ArcNotes

Welcome to your local markdown notebook.

## Start

- [ ] Choose or create a local note folder in the settings dialog.
- [ ] Create a note and write in plain text or Markdown.
- [ ] Use headings, lists, checkboxes, links, images, and code blocks to structure your notes.
- [ ] Organize notes with folders, subfolders, tags, and saved searches.

## Useful Basics

- [ ] Use the preview panel to check rendered Markdown.
- [ ] Use the note history buttons to move between recently opened notes.
- [ ] Open <Markdown Cheatsheet.md> to explore common Markdown syntax.
- [ ] Use local trash if you want deleted notes to be recoverable.
